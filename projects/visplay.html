Visplay

Sept. 2019 - Present

Visplay is an open-source project developed by the Colorado School of Mines ACM Chapter. It is a digital signage solution that makes improvement on the existing software controlling digital displays around Mine's campus

Although still a work in progress, major progress has been made. There exists custom rendering software, a custom distributed data store, and a simple web client to configure displays

My primary contribution has been developing the distributed data store. Due to the distributed nature of the system, there is no centralized data store. Rather, all data is stored in a CRDT (conflict-free replicated data type) on a Matrix server. I have done the work to develop specification and implementation of these CRDTs
